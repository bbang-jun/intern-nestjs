# intern-nestjs

- Node.js 버전: v20.11.0
- nest 버전: 10.3.2

- yarn으로 Nest.JS 설치
```bash
yarn global add @nestjs/cli
```

- Nest.JS 프로젝트 생성
```bash
nest new ./
```

- Nest.JS 프로젝트 실행
```bash
yarn run start:dev
```

- module 생성
```bash
nest g module {모듈명}
```

- controller 생성
```bash
nest g controller {컨트롤러명} --no-spec
```

- service 생성
```bash
nest g service {서비스명} --no-spec
```
